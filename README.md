# Финансовый трекер

Личный трекер расходов с тёмной темой. Все данные хранятся только у тебя — ничего не уходит в интернет.

## Что умеет

- Дашборд с тратами за день и месяц, прогресс по бюджету
- Добавление расходов с категорией, описанием и оценкой сомнения (0–5)
- История с фильтрами и экспортом в CSV
- Детализация: графики по дням, категориям, бюджет vs факт
- Настройка дохода, категорий и фиксированных расходов

---

## Быстрый старт

### 1. Установить Node.js

Скачай с [nodejs.org](https://nodejs.org/) (версия 18+).

### 2. Скачать и запустить

```bash
git clone https://github.com/dimaglobin/fintracker.git
cd fintracker
npm start
```

### 3. Открыть

Перейди на [http://localhost:3000](http://localhost:3000).

## Первая настройка

Приложение запускается с дефолтными категориями и нулевым доходом.

### Шаг 1: Укажи зарплату

Зайди в **Настройки** и введи ежемесячный доход. Лимиты по категориям распределятся автоматически — примерно так:

| Категория | Доля |
|-----------|------|
| Продукты | 20% |
| Еда вне дома | 7% |
| Такси | 4% |
| Транспорт | 5% |
| Здоровье | 3% |
| Развлечения | 5% |
| Одежда | 4% |
| Маркетплейсы | 4% |
| Красота | 2% |
| Прочее | 4% |

Итого ~58% дохода на переменные расходы. Остальное — на фиксированные (аренда, подписки) и накопления.

### Шаг 2: Подгони под себя

Поменяй лимиты категорий в **Настройки → Переменные расходы**. Добавь фиксированные расходы (аренда, ЖКХ, подписки).

### Шаг 3 (опционально): Спроси AI

Если не знаешь, какие лимиты поставить — скопируй этот промпт в ChatGPT, Claude или другой AI:

```
Помоги мне распределить бюджет. Мой ежемесячный доход: [СУММА] ₽.
Город: [ГОРОД]. Возраст: [ВОЗРАСТ]. Живу [один / с партнёром / семья].

У меня такие категории расходов:
- Продукты
- Еда вне дома (кафе, доставка)
- Такси
- Транспорт (метро, поезда)
- Здоровье (аптека, врачи)
- Развлечения (кино, концерты)
- Одежда
- Маркетплейсы (Ozon, Wildberries)
- Красота (стрижка, уход)
- Прочее

Мои фиксированные расходы: [АРЕНДА, ПОДПИСКИ, ЖКХ — или "пока не знаю"].

Предложи конкретные суммы в рублях для каждой категории.
Учти, что я хочу откладывать хотя бы 10-15% дохода.
Формат ответа — таблица: категория, сумма, % от дохода.
```

## Данные и приватность

- Все расходы хранятся **только на твоём компьютере** в папке `data/`
- Никакие данные не отправляются в интернет
- Очистить данные: **Настройки → Сбросить всё**
- Резервная копия: **История → Экспортировать CSV**
